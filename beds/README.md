# Furnitopia AR Bedden Experience

## Overzicht
Deze AR-functionaliteit stelt klanten in staat om bedden direct in hun eigen slaapkamer te plaatsen en te bekijken met hun smartphone camera.

## Technologieën
- **Model Viewer**: Google's AR viewer voor 3D modellen
- **WebXR**: Voor geavanceerde AR-ervaringen
- **AR.js**: Marker-based AR als backup

## Mappenstructuur
```
beds/
├── img/                 # Bedden afbeeldingen
│   ├── modern-bed.jpg
│   ├── classic-bed.jpg
│   ├── platform-bed.jpg
│   ├── bunk-bed.jpg
│   ├── king-bed.jpg
│   └── storage-bed.jpg
├── css/                 # Bedden-specifieke styles
├── js/                  # Bedden-specifieke JavaScript
└── README.md           # Dit bestand

├── modern-bed.glb
├── classic-bed.glb
├── platform-bed.glb
├── bunk-bed.glb
├── king-bed.glb
└── storage-bed.glb
```

## AR Functionaliteiten
- **Direct plaatsen**: Bedden direct op de vloer plaatsen
- **Echte afmetingen**: Zie exact hoe groot het bed is
- **360° bekijken**: Loop eromheen om alle kanten te zien
- **Kleuren testen**: Verschillende kleuren en stijlen uitproberen
- **Direct bestellen**: Bestel vanuit de AR-ervaring

## Browser Ondersteuning
- **Chrome/Edge**: Volledige AR ondersteuning
- **Safari**: AR via Quick Look
- **Firefox**: Basis 3D viewer
- **Mobiel**: Optimale ervaring op smartphones

1. Upload 3D modellen naar `/models/` map
2. Upload bedden foto's naar `/img/beds/` map
3. Test AR functionaliteit op verschillende apparaten
4. Optimaliseer 3D modellen voor snelle loading
